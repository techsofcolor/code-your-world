Code Your World

with Arduino

An interactive programming workshop for middle school students using the Arduino prototyping kit.


December 3rd 2016   12pm - 4pm

  Register

In Code Your World with Arduino, students will learn how they can use computers to interact with the physical world. By the end of class students will understand how many of the everyday things we encounter are programmable and think of new ways to program the world around them.



Goals and Objectives

    • Learn to build an electronic prototype using a sensor
    • Learn how sensors are programmed and used in the real world
    • Learn to work with others, as part of a team, to build functioning electronic prototypes

The Internet of Things ...
Is the next BIG thing.

From automatically turning on and off lights to detecting temperature changes students will create projects using the Arduino prototyping kit that will demonstrate the basics of electronics programming and introduce the "IoT" concept. It's only $10 and space is limited so, register your student now!


What is IoT??

Simply put, this is the concept of basically connecting any device with an on and off switch to the Internet (and/or to each other). This includes everything from cellphones, coffee makers, washing machines, headphones, lamps, wearable devices and almost anything else you can think of.
